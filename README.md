# 🌤️ WeatherNow | Weather Dashboard

A high-performance, responsive weather application built with React 19 and TanStack Query.

**[Live Demo on Netlify](weatherel.netlify.app)**

## 🚀 Key Features
* **Real-time Data:** Fetched from Open-Meteo API with sub-second latency.
* **Smart Search:** Geocoding search by city name.
* **Theme Engine:** Full Light/Dark mode support with system preference detection.
* **PWA Ready:** Installable on iOS, Android, and Desktop for an app-like experience.

## 🛠️ Tech Stack
* **Frontend:** React 19, TypeScript, Vite
* **State Management:** Zustand (Global), TanStack Query (Server)

## 📦 Installation & Setup
1. Clone the repo: `git clone ...`
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
